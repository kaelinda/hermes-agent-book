# 如何贡献 (CONTRIBUTING.md)

感谢你对 **Hermes Agent 中文 Start Book** 的兴趣！

本项目由 Hermes Agent 自我迭代维护，欢迎任何人以任何形式参与改进。

---

## 贡献方式

### 1. 提交 Issue
- 发现错别字、错误信息、技术不准确
- 提出新主题建议（如 Axolotl 微调指南、特定行业应用等）
- 功能请求（希望增加某个模板或案例）

### 2. 提交 Pull Request (推荐)
- 改进现有文档（结构优化、补充模板、更新信息）
- 新增章节或全新指南
- 美化排版、增加示例、修复链接
- 改进 README 或 GitHub Pages

### 3. 让你的 Hermes 参与贡献（最推荐方式）
使用以下 Prompt 让你的 Hermes Agent 帮忙：

```bash
hermes chat -q "请阅读本仓库中的 Hermes-Agent-中文-Start-Book-v2.0.md 和所有指南，对整个项目进行全面 review，找出可改进之处，然后使用 skill_manage 创建或更新相关 Skill，最后生成改进后的内容并给出 Pull Request 建议。"
```

---

## 贡献流程

1. Fork 本仓库
2. 创建你的分支 (`git checkout -b feature/awesome-improvement`)
3. 提交修改 (`git commit -m 'feat: 添加 xxx 改进'`)
4. 推送并创建 Pull Request

---

## 贡献规范

- 保持与现有文档一致的风格（深度 + 实战 + 模板 + 避坑）
- 新增内容请遵循“原理 → 模板 → 实战 → 避坑”的结构
- 使用中文写作，确保流畅自然
- 所有命令和 Prompt 必须可直接复制使用
- 欢迎在文档中增加你的 GitHub ID 作为贡献者
